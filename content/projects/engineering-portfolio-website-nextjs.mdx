## Overview

This is the site you are currently viewing: a content-first engineering portfolio designed for long-term maintainability and fast project updates.

## Problem / Constraints

- Needed one place to represent software, ME, and research work cohesively.
- Portfolio updates had to be low-friction and file-based (no CMS).
- Site needed to be performant, clean, and easy to extend.

## Approach

Implemented an App Router Next.js architecture with MDX project files, reusable components, category filtering/search, and a structured asset organization strategy.
