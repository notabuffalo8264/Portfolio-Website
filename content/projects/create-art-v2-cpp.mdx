## Overview

Create Art V2 is a follow-up to V1, redesigned around practical drawing interactions using mouse/stylus input plus keyboard shortcuts.

## Problem / Constraints

- V1 input model was limiting for detailed control.
- Needed faster, more intuitive drawing workflows.
- Project remained a self-driven learning build without AI-generated code.

## Approach

Rebuilt the interaction model in C++, implementing pointer-based drawing and shortcut-driven actions for mode/tool control.
