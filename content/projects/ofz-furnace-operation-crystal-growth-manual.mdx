## Overview

Created a dedicated operation and crystal-growth manual for the exact OFZ workflow used in my Bi2Se3 research. The manual captures setup sequencing, mirror/alignment handling, process flow, and practical guidance for day-to-day execution.

## Problem / Constraints

- Existing documentation did not match this specific ampule-based process configuration.
- Process reliability depended heavily on tacit knowledge and trial-and-error.
- New users lacked a single reference for startup, operation, and failure response.

## Approach

Structured the manual around the real workflow used in the lab:

- pre-run checks and setup steps
- operation flow and translational process handling
- common issues and corrective actions
- repeatability notes and best practices
