## Overview

Create Art V1 was my first custom digital art tool, where drawing was performed entirely through keyboard-driven controls.

## Problem / Constraints

- Needed to learn graphics and interaction fundamentals from scratch.
- Interface had to be usable without mouse support.
- Entire implementation was built as a personal learning exercise.

## Approach

Developed a JavaFX app with keyboard-mapped drawing actions, movement controls, and state updates to render user-generated patterns.
