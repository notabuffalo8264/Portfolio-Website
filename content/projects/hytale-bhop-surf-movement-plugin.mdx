## Overview

This project adds high-skill movement styles (B-Hop and Surf) to Hytale servers to create competitive, movement-focused game modes.

## Problem / Constraints

- Hytale plugin ecosystem is still evolving.
- Movement systems must feel responsive and predictable across server/client conditions.
- Requires careful tuning to balance accessibility with mastery.

## Approach

Implemented movement-state handling and velocity/acceleration logic for custom traversal behaviors, then iterated on feel and control windows.
